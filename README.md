# `ATIAM - WebAudio`

> Exercises for the ATIAM WebAudio course

## Install

```
npm install 
npm run dev
```

Your application will be accessible at `http://localhost:5000/`

## Available commands

#### `npm run dev`

launch the server and, watch the file system and transpile on save

#### `npm run build`

build the application

#### `npm run start`

launch the server
